### 控制之美

我们的生活其实每天都从控制开始：

* 控制自己跟随自己期望的轨迹，大脑自然而然地规划步长和频率，小脑将这些信息转换为对关节的控制；
* 控制自己拿起桌面上的一杯水，如何做到轨迹柔顺且平滑，如何定义自己的工作空间，我们的眼睛和触觉在这个过程中又是如何为我们规划一条collision-free的轨迹?
* 但是机器人，机器人又该如何通过更数学的语言，更清晰的逻辑结构，更简单统一的信息输入来为自己进行轨迹规划？
